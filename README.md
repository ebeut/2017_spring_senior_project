### Compile and Run: ###
1. [Install Node.js](https://nodejs.org/en/)
2. To check that npm is installed enter `npm -v` from the terminal
3. From the working directory in the terminal enter:
    - `npm install`
    - `npm run dev`
4. From a web browser navigate to "localhost:3000"
